# Overview
Specialised implementation of model railway block occupation using positional train detection at each end of block for UK four aspect colour light signalling written in Microchip MPASM assembly languge.
# Dependency
Requires [blcksqnc](https://github.com/MonitorComputing/blcksqnc).
